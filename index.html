Sobre Mim

Olá, sou o Pedro! Para mim, código é como uma caixa infinita de LEGO. Desde que descobri a programação, sou fascinado pela capacidade de juntar pequenos blocos de lógica para construir algo que resolve um problema real, que funciona e que impressiona.

Minha missão é criar essas estruturas digitais de forma sólida e elegante. Atualmente, estou focado em me especializar nas ferramentas mais robustas dessa "caixa de construção": o ecossistema Java com Spring Boot, ideal para erguer microsserviços que não só funcionam hoje, mas que estão prontos para crescer amanhã.

Fora do código, você pode me encontrar jogando , experimentando um bom café , ouvindo música ou imerso em um bom mangá, ou livro .

Minha Filosofia

Back-End Robusto e Escalável

Meu foco é construir o alicerce de aplicações. Utilizo Java e Spring Boot para criar APIs seguras, eficientes e preparadas para crescer, garantindo que a lógica de negócio seja sólida e confiável.

Código Limpo e Manutenível

Acredito que um bom código é aquele que é fácil de ler e dar manutenção. Aplico princípios como SOLID e Clean Code para criar software que não apenas funciona hoje, mas que é sustentável a longo prazo.

Automação para Eficiência

Processos manuais e repetitivos são oportunidades. Gosto de usar ferramentas como Docker para criar ambientes consistentes e scripts para automatizar tarefas, liberando tempo para focar em resolver problemas reais.
